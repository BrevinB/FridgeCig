# FridgeCig Instagram Content Pack — 30 Days, Ready to Post

Copy-paste captions, hashtags, and filming directions. Everything cross-posts to TikTok (post natively there too — same video, same caption, no watermark). Post cadence: **Mon / Wed / Fri** feed content, Stories on the in-between days. Days below map to that cadence over ~4.5 weeks.

Format ideas build on the pillars in the [original content strategy](../social-media-content-strategy.md); this pack turns them into finished, sequenced posts.

**Hashtag blocks** (paste under captions):
- **Block A (culture):** `#FridgeCig #DietCoke #DietCokeAddict #DietCokeTok #fridgecigarette #DietCokeLovers`
- **Block B (app/discovery):** `#FridgeCig #DietCoke #iOSApp #IndieApp #AppRecommendation #NicheApp`

**Before you start:** seed the demo account with the built-in screenshot data manager (Settings → Debug) so every screen recording shows a 14-day streak, badges, and friends instead of an empty app.

## Batch-filming guide (all 30 days in one ~2-hour session)

1. **Prep (15 min):** seed demo data, charge the watch, chill 4+ Diet Cokes, clean the fridge shelf (it's your set).
2. **Screen recordings (30 min):** capture ALL app footage in one pass — quick-log, badges, leaderboard, State Can map, weekly recap, milestone card, share-card editor, watch demo. That's Days 3, 8, 12, 17, 19, 22, 26, 29 done.
3. **Live-action (45 min):** film the fridge/face content — fridge cigarette open, POV one-left, "just drink water" stare, day-in-the-life inserts. Multiple takes, keep everything.
4. **Stills (15 min):** photograph cans/bottles/fountain cups for Story templates and carousel backgrounds.
5. **Edit as you go each week (not all at once):** 3 videos/week in CapCut using the captions above. Editing weekly keeps you able to jump on trending audio.

**Rule of thumb:** if a post flops, fine — it cost you 4 minutes. If a post pops, make 3 more in that format before the algorithm forgets you.
