# FridgeCig Marketing Kit

Everything here is built around two hard constraints: **1–3 hours per week** and **under $100/month**. The strategy is organic-first, with users' own share cards doing most of the marketing.

## The weekly playbook at a glance (~2.5 hours)

| Time | Task |
|------|------|
| 60 min (one sitting, e.g. Sunday) | Film/record 3 short-form videos from the content pack; caption and schedule them for the week (cross-post TikTok + Reels) |
| 30 min | Send **one** outreach email from the outreach kit (newsletter, creator, or podcast) |
| 30 min | Community: reply to comments, repost any #FridgeCig user content to Stories, drop one Reddit comment or X reply where Diet Coke is trending |
| 15 min | Check the scoreboard: App Store Connect downloads by source + RevenueCat trials; note what post performed best |

## Do-this-first checklist

- [ ] Put the live App Store link in `docs/index.md` (it still says "coming soon") and everywhere else (IG bio, TikTok bio, X bio)
- [ ] Claim **@fridgecig** on Instagram, TikTok, and X if not already done; convert IG to a professional account
- [ ] Set up a link-in-bio page (or link straight to the App Store — one tap beats a menu)
- [ ] Use the built-in screenshot seeder (`DietCokeTracker/Debug/ScreenshotDataManager.swift` via Settings → Debug) to generate a good-looking account, then capture the shot list in the [original strategy doc](../social-media-content-strategy.md#screenshot--video-shot-list)
- [ ] In App Store Connect: create 2–3 **custom product pages** (one per channel: `social`, `reddit`, `press`) so you can tell where installs come from without an attribution SDK
- [ ] Create **offer codes** in App Store Connect for creator seeding (free Pro), one code batch per creator so redemptions are traceable
